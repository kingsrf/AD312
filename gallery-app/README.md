# Recipe Gallery

A simple React + Vite application showcasing an interactive recipe gallery. Users can browse recipe images with Next and Previous buttons.

### Features

- Displays recipe images with descriptions
- Navigates using Next and Previous buttons
- Prevents navigation beyond the first or last image
- Uses React state with the useState Hook
- Includes tests for normal and edge cases

#### Tech Stack

- React
- Vite
- JavaScript
- CSS
- React Testing Library
- Jest

#### Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:
```bash
npm run dev
```

Open the local URL shown in the terminal.

#### Testing

Run tests:
```bash
npm test
