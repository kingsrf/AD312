# React Counter App

A simple React + Vite counter application demonstrating state management, asynchronous state updates, batching, and React state as a snapshot.

## Features

- Increment count by 1
- Increment count after a 2-second delay
- Demonstrate React batching with repeated state updates
- Correctly increment twice using updater functions

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:
```bash
npm run dev
